## How to Run Locally

- Prerequisites: [Node.js](https://nodejs.org/en/) and [Next.js](https://nextjs.org/)

### Installation

```bash
  npm install
  npm run init-db
  npm run dev
```
It should be running on [http://localhost:3000](http://localhost:3000)
